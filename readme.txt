These are tests for a dummy slot game. 
Game is showing different text messages depending on how many tiles has match.

Tests are written with a Cucumber BDD framework, using WebdriverIO and everything runs under Chromedriver.
You can find all test scenarios in './test/features/game.feature'.
The actual steps realization is in './test/features/step_definitions/chrome.steps.js'.

To run this on your local machine:

1. Install all dependencies with 'nmp install'
2. Start the game with 'nmp start'
3. Run tests with 'npm test'
